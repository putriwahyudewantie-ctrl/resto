# Resto Web Application (Laravel) – Group Project

## 📌 Project Overview
This project is a group assignment developed using the Laravel framework. It is a simple restaurant web application designed to understand how a real-world web system is structured using MVC architecture.

The application was built as part of academic learning to practice collaborative software development and web application fundamentals.

---

## 🛠️ Technologies Used
- Laravel (PHP Framework)
- PHP
- MySQL Database
- Bootstrap
- HTML
- CSS

---

## 🎯 Features
- Simple restaurant landing page
- Menu display system
- Basic web layout using Bootstrap
- MVC structure implementation (Model, View, Controller)
- Database integration (MySQL)
- Basic routing system in Laravel
