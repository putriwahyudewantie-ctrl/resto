# Resto Web Application (Laravel) – Group Project

## 📌 Project Overview
This project is a group assignment developed using the Laravel framework as part of our learning process in web development. The main goal of this project is to understand how a web application is structured using the MVC (Model, View, Controller) concept.

The application is a simple restaurant website that displays menu information and demonstrates basic backend and frontend integration.

---

## 🛠️ Technologies Used
- Laravel (PHP Framework)
- PHP
- MySQL
- Bootstrap
- HTML
- CSS

---

## 🎯 Project Features
- Restaurant landing page design
- Menu display system
- Basic CRUD functionality (Create, Read, Update, Delete)
- MVC structure implementation using Laravel
- Database integration using MySQL
- Responsive UI using Bootstrap
